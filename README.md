## 🎯 What This Project Does

EdUPro serves 6 types of pediatric professionals — but every user gets the same course recommendation. This project fixes that.

A machine learning pipeline that:
1. Segments 2,000 professionals into 4 behavioral archetypes using K-Means clustering
2. Delivers personalized CPD course recommendations through a hybrid scoring engine
3. Presents everything in a 6-page interactive Streamlit dashboard for EdUPro management
